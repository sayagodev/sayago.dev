## Description

Institutional website for Zihuame Mochilla, a non-profit organization that has been working with indigenous communities since 2003. It includes a donation system with one-time payments and monthly subscriptions, integration with Stripe, CMS with Sanity for content management, and informative sections on its areas of intervention, events, partners, annual reports, and transparency.

## Roles

UI/UX, Development

## Technologies

Figma, Tailwind, Next.js, Sanity CMS, Stripe, GSAP, TypeScript

## Objectives

- Design and develop an institutional website that presents the organization's mission and work.
- Implement a donation system with one-time payments and monthly subscriptions using [Stripe](https://stripe.com/).
- Integrate a CMS ([Sanity](https://www.sanity.io/)) so that the team can manage content without technical knowledge.
- Create a seamless user experience with animations and responsive design.
