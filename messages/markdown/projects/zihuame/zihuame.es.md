## Description

Sitio web institucional para Zihuame Mochilla, A.C. que trabaja con comunidades indígenas desde 2003. Incluye sistema de donaciones con pagos únicos y suscripciones mensuales, integración con Stripe, CMS con Sanity para gestión de contenido, y secciones informativas sobre sus áreas de intervención, eventos, aliados, reportes anuales y transparencia.

## Roles

Full Stack Development, UI/UX

## Technologies

Figma, Tailwind, Next.js, Sanity CMS, Stripe, GSAP, TypeScript

## Objectives

- Diseñar y desarrollar un sitio web institucional que presente la misión y trabajo de la organización.
- Implementar un sistema de donaciones con pagos únicos y suscripciones mensuales mediante [Stripe](https://stripe.com/).
- Integrar un CMS ([Sanity](https://www.sanity.io/)) para que el equipo gestione contenido sin conocimientos técnicos.
- Crear una experiencia de usuario fluida con animaciones y diseño responsive.
